1. User Data Collection

NutriSnap-X collects only the minimum required data such as user name, email, selected health goal, and food logs entered by the user.

2. Purpose of Data Use

Collected data is used strictly for nutrition analysis, personalized recommendations, and dashboard analytics. No data is used for advertising or resale.

3. AI Processing

Uploaded food images are processed by an AI model only for food recognition and nutrition estimation. Images are not shared with third parties.

4. Data Storage

User records are stored in a secured database structure with user-level isolation. Each user can access only their own data.

5. Security Measures

Passwords are stored in hashed form. Authentication is required to access dashboards and reports.

6. Academic Project Notice

NutriSnap-X is developed as an academic project prototype. It is not a medical diagnosis system.

7. User Control

Users may stop using the system at any time. Future versions may include data deletion options.
